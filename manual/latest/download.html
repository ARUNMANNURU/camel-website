Getting the Binaries using Maven 2

To use this release in your maven project, the proper dependency configuration that you should use in yourMaven POMis:

<dependency>
  <groupId>org.apache.camel</groupId>
  <artifactId>camel-core</artifactId>
  <version>2.20.2</version>
</dependency>
