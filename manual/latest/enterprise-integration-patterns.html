Enterprise Integration Patterns

Camel supports most of theEnterprise Integration Patternsfrom the excellent book byGregor HohpeandBobby Woolf.

If you are new to Camel you might want to try the Getting Started in the User Guide before attempting to implement these patterns.

The EIP icons library is available as a Visio stencil file adapted to render the icons with the Camel color : sand. Download itherefor your presentation, functional and technical analysis documents. The original EIP stencil is also available inOpenOffice 3.x Draw(thanks to Marco Garbelini) ,Microsoft Visio, orOmnigraffle.
