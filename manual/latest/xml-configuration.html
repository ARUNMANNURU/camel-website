XML Configuration

We recommend developers use the JavaDomain Specific Languagewhen writing routing rules as this provides maximum IDE completion and functionality while being the most expressive. However if you wish to put your routing rules in XML you can via the Camel XML language.

Camel XML usesSpring2 namespaces; so that you can configure your routing rules within yourSpring XML configuration file; you can also useJava code to configure components and endpoints.

For examples on how to use Camel XML, see theEnterprise Integration Patternsor refer to theSpring Support.
