Endpoints

Camel supports the Message Endpoint pattern using theEndpointinterface. Endpoints are usually created by a Component and Endpoints are usually referred to in the DSL via their URIs.

From an Endpoint you can use the following methods

  • createProducer()will create aProducerfor sending message exchanges to the endpoint

  • createConsumer()implements the Event Driven Consumer pattern for consuming message exchanges from the endpoint via aProcessorwhen creating aConsumer

  • createPollingConsumer()implements the Polling Consumer pattern for consuming message exchanges from the endpoint via aPollingConsumer
